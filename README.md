 Análisis de Eficiencia Hospitalaria con Power BI.

Dashboard interactivo desarrollado en Power BI para analizar la eficiencia operativa y costos de un hospital ficticio con 10,000 pacientes.

 📌 Descripción del Proyecto

El objetivo de este proyecto es demostrar habilidades en:
- Modelado de datos en Power BI.
- Transformación de datos con Power Query.
- Medidas DAX para cálculo de KPIs y análisis de costos.
- Visualización de datos con gráficos interactivos.

🔍 PRINCIPALES HALLAZGOS
El análisis de 10,000 pacientes reveló que Diabetes es la condición más costosa, especialmente en jóvenes de 18 a 35 años, quienes representan el segmento con mayor costo promedio ($26,159). Asma es la condición más común, y Obessidad es la segunda más costosa, afectando principalmente a adultos. Recomiendo implementar programas preventivos enfocados en estos segmentos para reducir costos y mejorar la eficiencia operativa del hospital.
